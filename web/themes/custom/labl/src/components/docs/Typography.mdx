# Typography

We use fluid type scale for font sizes, the font size will resize dynamically according to the screen width

## Headings

All heading styling is defined on the class `.h2`, not the element `<h2>`, this is imperitive for SEO and Accessibility.


This is to ensure heading order is kept in sequence, even when design styles do not represent it.

e.g: Card Titles are most likely to be a `<h3>`,  but are usually styled similar to `<h5>`, so we want to be able to use `<h3 class="h5">Card Title</h3>`
